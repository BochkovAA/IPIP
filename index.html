JavaScript для начинающих

Что такое JavaScript?

JavaScript — это язык программирования, используемый для создания интерактивных элементов на веб-страницах. Он позволяет оживлять страницы, добавляя динамические возможности, такие как обновление контента, анимации и многое другое.

Для чего используется JavaScript?

  • Создание интерактивных элементов на веб-страницах.
  • Разработка веб-приложений и игр.
  • Работа с браузерным API, например, для создания всплывающих окон или обработки событий пользователя.
  • Обмен данными с сервером без перезагрузки страницы (AJAX).

Когда JavaScript не пригодится?

JavaScript не используется для создания статических страниц, а также для задач, требующих высокой производительности на уровне системы, таких как разработка операционных систем или работа с аппаратным обеспечением. Для таких задач лучше подходят языки, как C++ или Rust.

Демонстрация задач

Задача 4 из первой практической

Запросить у пользователя число больше 5 и выдать сообщение:

 
let number;
do {
    number = prompt('Введите любое число больше 5:');
    if (number === null) {
        alert('Вы отменили ввод');
        break;
    } else if (!isNaN(number) && +number > 5) {
        alert('Поздравляем! Вы ввели число больше 5.');
    } else {
        alert('Ошибка! Введите число больше 5.');
    }
} while (number !== null && (+number <= 5 || isNaN(number)));
