О JavaScript

JavaScript - это язык программирования, который широко используется для создания интерактивных веб-сайтов. Он позволяет добавлять динамическое поведение к веб-страницам.

Вы можете использовать JavaScript для валидации форм, создания анимаций, обработки событий и многое другое.

JavaScript также позволяет взаимодействовать с элементами HTML и CSS на странице, делая сайты более интересными и функциональными.

Однако, для создания сложных приложений, часто требуется знание других технологий и фреймворков.

Практические задания

Практическая 1, задание 4


let number;
do {
    number = prompt('Введите любое число больше 5:');
    if (number === null) {  
        alert('Вы отменили ввод');
        break;
    } else if (!isNaN(number) && +number > 5) {
        alert('Поздравляем! Вы ввели число больше 5.');
    } else {
        alert('Ошибка! Введите число больше 5.');
    }
} while (number !== null && (+number <= 5 || isNaN(number)));

Практическая 2, задание 3


let array = [5, 10, 15, 20, 25, 30, 35, 40];
let container = document.getElementById('array-container');

array.forEach((item, index) => {
    let div = document.createElement('div');
    div.className = 'item';
    div.textContent = `Элемент ${index + 1}: ${item}`;
    div.setAttribute('data-length', `Длина массива: ${array.length}`);
    container.appendChild(div);
});
